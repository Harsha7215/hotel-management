Hotel Management System (Full Stack)
A sophisticated web-based solution designed to streamline hospitality operations, manage room bookings, and enhance user experience. This project leverages Django for backend logic and Modern Web Technologies for a responsive, interactive frontend.

Key Features

Role-Based Access Control (RBAC): Developed secure modules for user registration and login with different access levels for guests and administrators.

Dynamic Room Booking: Engineered complex logic to manage room availability and reservations using the Django MVT architecture.

Real-time Admin Dashboard: Implemented a centralized hub for administrators to monitor occupancy rates and manage bookings in real-time.


Database Optimization: Designed and optimized schemas using Django ORM and SQL to ensure efficient data relationships.


Responsive UI: Built a user-friendly interface using HTML5, CSS3, and JavaScript to ensure seamless performance across mobile and desktop devices.


Technical Stack

Backend: Python, Django (MVT Architecture).


Frontend: HTML5, CSS3, JavaScript.


Database: SQL with Django ORM for data persistence.


Tools: Git, GitHub, and Linux for development and version control.


Project Structure
/models: Defines the database schema for Rooms, Users, and Bookings.

/views: Contains the logic for processing requests and handling the booking workflow.

/templates: Responsive HTML files for the guest interface and admin dashboard.

/static: CSS and JavaScript files for styling and interactivity.


Installation & Setup
Clone the Repository:

Bash
git clone https://github.com/your-username/hotel-management-system.git
Install Dependencies:

Bash
pip install django
Run Migrations:

Bash
python manage.py makemigrations
python manage.py migrate
Start the Server:

Bash
python manage.py runserver
